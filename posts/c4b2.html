Windows常用的100个快捷方式

分享
技术
发布日期:   2024-09-08

阅读时长:   2 分

基础

Ctrl+Z : 撤销
Ctrl + A : 全选
Ctrl + C : 复制
Crtl+Insert: 复制
Ctrl + V : 粘贴
Shift+Insert：粘贴
Ctrl + X : 剪切
Alt + Tab : 切换窗口
shift+alt+tab：alt+tab反过来    
Ctrl + Alt + Del : 任务管理器界面
Alt + F4：关闭窗口
Ctrl+W : 关闭窗口
Ctrl + F : 查找
Ctrl +H : 替换
Ctrl + N : 新建空白窗口
Ctrl + 滚轮 : 缩小放大
Ctrl + Y：恢复撤销
shift+方向键：选择文本
shift+delete：永久删除
delete：删除
Ctrl + Shift + Esc ：打开任务管理器
ctrl+P：打开打印菜单
ctrl+D：把文件移入回收站
ctrl+S：保存
Ctrl+R：刷新

高级

Win：开始菜单
Crtl+Esc：打开开始菜单
Win + L: 锁定计算机
Win + D : 回到桌面
Win + Tab : 任务视图
Win + C : 打开小娜
Win + Prt Sc : 全屏截图
Win + I : 打开设置
Win + S : 搜索
Win+Q：搜索
win+F2 : 重命名
win+F5 : 刷新
Win + X : 右键windows标的菜单
Win + V : 剪贴板
Windows + . ：emoji
Win+；：emoji
Ctrl + Windows + Shift + B：重启图形驱动
Ctrl + Shift + 点击任务栏图标 ：以管理员权限打开
Windows + K ：快速连接设备
Ctrl + 点击任务栏上同应用的多个窗口 ：快速切换程序窗口
Windows + A ：打开通知中心
Windows + 左方向键：向左贴靠
Win+右方向键
Win+上方向键：窗口最大化
win+shift+上方向键：窗口高度最大
Win+下方向键：移动窗口
Win + Shift + S：截图工具
PrtScr ：截图工具
Win + Alt + R ：游戏录制
Win + G ：Windows游戏栏
Win + 1-9：打开任务栏应用
Win+alt+1-9：右键点击任务栏图标
Win + Space：切换输入法
alt+shift：切换输入法
Win + P ：更改投影模式
Win + R ：运行
Win + Alt + D ：打开日历
Win + , ：看桌面
Win+M: 最小化所有应用
Win + Shift + M：恢复最小化应用
Win+home：除前台应用所有程序最小化
Win + Shift + 左方向/右方向：把窗口放到另一个屏幕
Win+W：打开Windows ink工作区
Win+B：选择任务栏小图标

文件管理器

Win+E：打开文件管理器
backspace：返回上一级
Alt + P ：预览图片
Alt + Enter ：打开属性
alt+双击左键：打开属性
Ctrl + Shift + N：新建文件夹
Alt+D：选择路径栏

Shift +点击任务栏图标 ：打开新窗口
Win + T：选择任务栏应用
Win + Ctrl + D：新建虚拟桌面
Win + Ctrl + Left/Right Arrow ：切换虚拟桌面
Win + Ctrl + F4 ：关闭虚拟桌面

浏览器

Alt + Left Arrow：返回上一页
Alt + Right Arrow：前进一页
Ctrl + 1-9 ：切换标签页
ctrl+tab：切换标签页
f6：选择地址栏网址
f3：页内搜索
f11：全屏模式
ctrl+L: 选中浏览器地址栏
Home：转到页首
End：转到页尾
鼠标中键点链接：新标签页打开    

win+U：轻松使用设置
win+加号：打开放大镜
win+减号：缩小
Win+H：打开听写功能
Win+F1:win10在线帮助
